# EMRE — Comic Portfolio (Next.js)

Çizgi roman temalı frontend developer portföyü. Orijinal DC HTML sürümünün Next.js App Router yeniden yazımı.

## Özellikler

- TR / EN dil geçişi (wipe animasyonu)
- Sayfa loader (POW!)
- Sticky navigasyon
- CSS ile çizilmiş avatar karakter
- Marquee şerit
- Hakkımda panelleri
- Proje kartları + modal detay
- Yetenek barları
- Deneyim zaman çizelgesi
- İletişim bölümü

## Geliştirme

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) adresini açın.

## Build

```bash
npm run build
npm start
```

Orijinal dosyalar `_legacy/` klasöründe saklanır.
